# Backend

E-commerce website backend API with Spring Boot

## Requirements

- Java 17
- Maven 3.8
- PostgreSQL 16

## Tools

- Spring Boot 3.x
- Spring Security
- Spring Data JPA
- PostgreSQL

## Run Development

```bash
# Configure database in application.properties first
mvn clean install
mvn spring-boot:run
```

Server will start at [http://localhost:8080](http://localhost:8080)
